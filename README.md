# Clocktower (Builder Timer)

A sleek, multi-account builder tracker for Clash of Clans. Monitor upgrade timers across all your villages in a single, unified dark-mode dashboard.

## Features

- **Multi-Account Support**: Track builders for multiple villages simultaneously.
- **Unified Dashboard**: See all active upgrades across all accounts in one view.
- **Dark Mode**: Fully optimized dark theme for night-time clashing.
- **No Login Required**: All data is stored locally in your browser.
- **Responsive Design**: Works great on both desktop and mobile.

## Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
